## 🏛️ Professional Legal Dashboard

JurisFlow is a modern, feature-rich legal case management system built with React, Tailwind CSS, and Vite.

## ✨ Features

### 📋 Case Management
- Create, Read, Update, Delete cases
- Status tracking (Active, Pending, Closed)
- Priority levels (Low, Medium, High, Urgent)
- Document and hearing tracking
- Amount tracking

### 👥 Client Management
- Client profiles with contact details
- Case history and tracking
- Status management

### 📅 Calendar
- Event scheduling
- Hearing and meeting tracking
- Monthly calendar view

### 📊 Reports & Analytics
- Case statistics
- Priority distribution
- Case type analysis
- Recent activity tracking

## 🛠️ Tech Stack

- **React 18** - UI Framework
- **Tailwind CSS** - Styling
- **Vite** - Build Tool
- **React Icons** - Icon Library
